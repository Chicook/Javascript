# Inmutabilidad

# 10 tipos de datos
* Existen primitivos y complejos

# Primitivos
* string (cadenas de texto)
* number (numeros)
* boolean (true/false -> verdadero/falso)
* null (valor nulo )
* undefined (indefinido)
* symbol (simbolos, elementos que son unicos)
* bigint (alojar numeros muy grandes)

# Complejos
* object (objetos)
* array (objetos)
* function (funciones)
